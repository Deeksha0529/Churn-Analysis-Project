# Churn-Analysis-Project
End-to-end OTT Churn Analysis pipeline built with Python (Pandas, NumPy, Seaborn) &amp; SQL (SQLite3). Analyzed multi-table subscriber, plan &amp; support ticket data across 20+ KPIs. Uncovered a 28.6% churn rate, highlighted a 55.6% monthly contract churn spike, and quantified $2,047 in CLTV loss to drive customer retention.
